Saturhack

Saturday Coding in a Collaborative Atmosphere

When: Saturday, September 20th 1pm - 6ish

Where: The Network Hub Downtown Vancouver

How To Get There: Take the SkyTrain to Waterfront Station station and get off.

Saturhack is a weekend gathering of Web Developers looking for a collaborative atmosphere to work on their personal projects. We love to code but find it's boring to do alone sometimes. So we hit up some friends on Twitter to see if anyone wanted to hack their code in a shared space, and Saturhack was born! Whether you're working in HTML, Javascript, PHP, Ruby, or Python, you are welcome to join us.

Since it's the weekend, beers are also a welcome part of Saturhack. Bring a brew or grab one at the liquor store downstairs and code with a cold one.*
